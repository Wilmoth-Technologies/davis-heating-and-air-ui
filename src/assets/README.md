# Brand Images

Place the brand logo images in this directory:

- `friedrich.png` - Friedrich HVAC brand image
- `rheem.png` - Rheem HVAC brand image  
- `gree.png` - GREE HVAC brand image

The images should be:
- PNG format (or JPG/WebP)
- Optimized for web use
- Recommended size: 400-600px width for best quality
